== Installation ==

There are two ways to install Standard: via the WordPress Dashboard and via FTP.

Using the WordPress Dashboard...

1.	Locate the 'Appearance' menu
2.	In the 'Appearance' menu, click on 'Themes'
3.	On the 'Themes' page, click on the 'Install Themes' tab
4.	Click on the 'Upload' link at the top of the page
5.	On the 'Upload' page, locate the theme archive on your machine and click 'Install Now'
6. 	On the following page, click 'Activate'

Using FTP...

1.	Unzip standard.zip on your local machine
2.	Connect to your server via FTP
3.	Locate the `/wp-content/themes/` directory
4.	Upload the `standard` directory to your server
5.	In the WordPress Dashboard, navigate to the 'Themes' page under the 'Appearance' menu
6.	Locate 'Standard' and click 'Activate'
