Admin Panel

Make decisions, not choices

Global

FeedBurner URL

Enter your FeedBurner URL and click Save Changes.

Your WordPress site comes with an RSS feed built in to help visitors subscribe to updates. Many of our customers choose to create an RSS feed with FeedBurner instead of using the native one provided because it provides detailed statistics, among other advantages.

Google Analytics

Enter your Google Analytics ID and click Save Changes.

You have two excellent options for collecting statistics about your site’s traffic, and both are free. The light and convenient option is called WordPress.com Stats and it’s found in the Jetpack suite of plugins, created by the makers of WordPress. The more complicated, but much more comprehensive and detailed option is Google Analytics.

Site Mode

Select Online or Offline and click Save Changes.

WARNING: Taking site offline will hide all content from site visitors and search engines. This could significantly impact your ranking in search engines.

Online

When your site is in online mode, it will function like any other WordPress site.

Offline

When your site is in offline mode, visitors will not be able to see posts or pages. They will only see the message you configured. Make sure you are logged out of the admin panel to experience what site visitors can see.

If you log into the admin panel, you will be able to see the working version of your site. You can make changes and preview them, while keeping the changes private from the public at large. This is especially handy if you are creating a new site or performing maintenance on an existing site.

Offline Message

Enter the public message for site visitors then click Save Changes.

This option only displays when site mode is set to ‘offline’.

Presentation

Logo

Click upload, select an image file to upload, then click Save Image.

When you upload a logo it replaces the site title and tagline in the site header. Delete the logo and they will once again be displayed.

The maximum width for logos is 300px.

Site Icon

Click upload, select an image file to upload, then click Save Image.

The site icon is displayed in the address bar of web browsers. Some mobile devices will use it as an app icon if a bookmark to your site is saved to the launch screen.

Preferred dimensions are 144px x 144px.

Contrast

Change the contrast to Dark or Light then click Save Changes.

The main function is to switch the contrast of text so that it’s readable in relation to customized backgrounds. You can change your site’s background to either a solid color or an uploaded image in Appearances -> Backgrounds, but depending on the color or image selected, you could end up with dark text against a dark background making it impossible to read.

In these cases, switch the contrast.

Dark (Default)

Light

Sidebar Orientation

Select Left Sidebar, Right Sidebar, or No Sidebar / Full Width and then click Save Changes.

This affects the layout of all pages and posts on the site. Move the sidebar to the left or right, or remove it completely and allow you content to be displayed at the “full width” of the page.

NOTE: There is an option to override this setting on each post and page.

Display Breadcrumbs

Select Always or Never then click Save Changes.

For Users

The term Breadcrumbs refers to a hierarchical list of links, usually filled with categories and post titles, that are presented on every page so that visitors can find and access other categories, pages, and posts in the hierarchy.

For SEO

Search engines favor sites with Breadcrumbs turned on because these hierarchical lists help the search engines categorize and organize your site’s content in their database.

Display Featured Images

Select Always, Never, On index only, or On single posts only and then click Save Changes.

Featured images are thumbnail images in the post title. You can choose to always display them on both the post list and on the single post page, on the post list (index) only, on single posts only, or not at all in either location.

Display Footer Credits

Select Always or Never and then click Save Changes.

Footer credits are are everything after the copyright and site name in the footer.

— Lean by Jason Bradley

Social

Font Awesome

Uses the brand icons from Font Awesome.

Active Icons

Arrange icons in the order in which you want them displayed in the menu bar.

Drag an icon back to the library to remove it from the menu bar.

Click an icon to set it’s destination URL.

The URL for the RSS icon is read only and can not be updated. If the native WordPress RSS feed is being used, it will show the local URL. If a FeedBurner feed has been configured in the Global section, it will show the FeedBurner URL.

Due to space limitations, Lean looks best with 7 icons or less.

Icon Library

To use a social icon in your header menu bar, drag it out of the Icon Library and into Active Icons.

The Icon Library comes with 14 social network icons.

Reset Defaults

If you want to remove all of the active icons and their URL, simply click the reset defaults link.

Publishing

Display Author Box

Select Always or Never and then click Save Changes.

The author box displays key information about the author of a blog post such as:

  • Author Image (Thumbnail)
  • Post Archive Link
  • Twitter Link
  • Facebook Link
  • Google+ Link

Author Image (Gravatar)

Create a Gravatar account using the same email address that is in your user profile.

Gravatar is a free service that allows you to configure a personal avatar image that “follows you around” as you post comments on blogs and sites throughout the Internet. As long as you use the email address associated with your Gravatar account, you’ll have an image of you (or your brand) when you leave comments.

To configure an Author Image in the Author Box you will need to create a Gravatar account using the same email address that is configured in your user profile on your site.

Post Archive Link

This is automatically generated per author, no configuration required.

Twitter, Facebook, and Google+ Links

These are configured for each author in their user profiles. To change yours, in the left navigation of the WordPress admin panel go to Users -> Your Profile.

Privacy Policy

Click the Generate button, edit the new Privacy Policy in the Pages section, then Publish the page.

It is an act of good web citizenship for all site owners to clearly state their position on visitor privacy. Lean makes getting started simple. We have created a draft Privacy Policy for you to edit and customize to your needs.

When you publish the privacy policy a link will be added to your footer.

IMPORTANT: Your Privacy Policy is not complete until you edit it for accuracy and publish the page.

Comment Policy

Click the Generate button, edit the new Comment Policy in the Pages section, then Publish the page.

It is also an act of good web citizenship for site owners to clearly state their position on visitor comments. We have created a draft Comment Policy for you to edit and customize to your needs.
