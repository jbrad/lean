Page Templates

Give your pages a little bit of love

Default

This is the default layout for pages. It includes the main content area and sidebar. You can change the location of the sidebar at Lean -> Presentation section

Full Width

This layout will remove the sidebar for pages.

Archives

This layout will print out the archives for all of your posts, pages, and categories. This includes a sidebar by default.

Sitemap

This section will show your authors, pages, and posts.

Jumbotron

This will show the page title and content inside the header (jumbotron) area. It will also show the marketing widgets below it, and the featurettes if you have any created.

Image Slider

This will show the Slides in the header. It will also show the Marketing Widgets below it, and the Featurettes if you have any created.

Marketing Widgets

These are sidebar areas that you can add any widget to.

Used for both the Image Slider and Jumbotron page templates.

Custom Post Types

There are two custom post types that come with Lean, Slides and Featurettes.

Slides

Used on the Image Slider page template.

The images will show as you have uploaded them. You should make sure the dimensions are the same for each image.

Includes the following:

  • Title .carousel-caption h3
  • Caption .carousel-caption p
  • Indicators .carousel-indicators
  • Next Slide .carousel-control.left
  • Previous Slide .carousel-control.right

Featurettes

Used for both the Image Slider and Jumbotron page templates.

The featurette image and content will alternate from left to right to create a zip zag pattern.
